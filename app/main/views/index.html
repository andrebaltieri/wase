What is WASE?

WASE is a complete modern web development stack that runs on Windows and Mac, helping you to achieve your development goals faster and organized.

WASE is the acronym for WebAPI, AngularJS, SQL Server and Entity Framework, a very known stack for Microsoft developers, and now, for open source community.

With WASE, you can easily create smart, robust and lightweight applications persisting your data In Memory, in a SQL Server or other database you prefer.

Tools for Non-Windows Developers

If you're running WASE STACK on Windows, we highly recommend you to download and install visual studio 2013 community.

If you're running WASE STACK on Mac OS, we recommend you to read the Mac OS Getting Start Guide to obtain more information about our recommended tools.
